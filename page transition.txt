The user loves the animations on the HomePage but feels other pages (Matrix, KPI, etc.) are too static/stiff.

Please create a reusable animation wrapper component: `components/ui/page-transition.tsx`.

**Requirements:**
1.  Use `framer-motion`.
2.  It should wrap the page content.
3.  **Animation:**
    - Initial: `opacity: 0`, `y: 20` (slightly down).
    - Animate: `opacity: 1`, `y: 0`.
    - Transition: `easeOut`, duration 0.4s.
4.  **Stagger:** If it has children, apply `staggerChildren: 0.1` so items appear one by one.

**Export:**
- `PageHeader`: For the title area (fade in first).
- `PageContent`: For the main grid/table (fade in second with delay).
